  • Trajectory Planning algorithm for autonomous racing car

    Year: 2024/25

    Implemented a SOTA trajectory planning algorithm in C++ for an autonomous Formula Student car as an optimal control problem

    Approach developed in Simulation by alumnus in Formula Student Team

    Uses acados C++ Solver to generate time-optimal and jerk-minimal trajectory on track with constraints on acceleration profile

    Algorithm validated in a ROS-integrated control-in-the-loop simulation and field-tested at the Formula Student competition in Czech Republic

    Technologies used: C++, Python, acados, CasADi, ROS

  • Path Planning algorithm for autonomous racing car

    Year: 2023/24

    Implemented a path planning algorithm in Python for an autonomous formula student car using delaunay triangulation

    Algorithm successfully field-tested and validated at the Formula Student competition in Italy

    Later ported into C++ for a drastic performance uplift: ~20ms to less than 1ms

    Technologies used: Python, NumPy, SciPy, ROS, C++
