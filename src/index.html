Projects

Here are some of the projects I've worked on:

  • Trajectory Planning algorithm for autonomous racing car

    Year: 2024/25

    Implemented a SOTA trajectory planning algorithm in C++ for an autonomous Formula Student car as an optimal control problem

    Approach developed in Simulation by alumnus in Formula Student Team

    Uses acados C++ Solver to generate time-optimal and jerk-minimal trajectory on track with constraints on acceleration profile

    Algorithm evaluated in Formula Student Event FSCzech

    Technologies used: C++, Python, acados, CasADi, ROS

  • Path Planning algorithm for autonomous racing car

    Year: 2023/24

    Implemented a path planning algorithm in Python for an autonomous formula student car using delaunay triangulation

    Algorithm evaluated in Formula Student Event FSItaly

    Later ported into C++ for a drastic performance uplift: ~20ms to less than 1ms

    Technologies used: Python, NumPy, SciPy, ROS, C++

  • Machine Learning Model for Predictive Maintenance

    Year: 2024

    Pair project with a fellow student, results were compiled in a graded paper

    Built and trained a machine learning model to predict the remaining lifetime of a machine.

    Used EDA and feature selection as well as feature engineering algorithms to create an optimized dataset

    Model was trained using ensemble methods such as Random Forest and evaluated using cross validation

    Additional testing with comparisons between ML Models were done to determine best Model for use case

    Technologies used: Python, scikit-learn, TensorFlow, pandas, NumPy

    Github repository: Repository

  • Unsupervised Learning Model to Create a Preventive Maintenance Plan for Delivery Service Packing Stations

    Year: 2024

    Project using real company data to create and evaluate new maintenance plan, results were compiled in a graded paper.

    Using Big Data methods to aggregate data from different sources and perform clustering and PCA to sort all stations into groups with similar maintenance needs

    Technologies used: Python, scikit-learn, pandas, NumPy

    Github repository: Repository
